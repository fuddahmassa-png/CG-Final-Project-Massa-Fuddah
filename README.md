# مشروع بيانيات الحاسوب - Sky Runner

فكرة اللعبة
لعبة Sky Runner ثلاثية الأبعاد تعتمد على القفز فوق المنصات وجمع النجوم مع تجنب السقوط في الفراغ.

### عناصر التحكم (Controls)
* **W, A, S, D**: للحركة في الاتجاهات الأربعة.
* **Space (المسطرة)**: للقفز.
* **R**: لإعادة اللعب فوراً عند الخسارة (Reset).
* **Esc**: للخروج من اللعبة.

* تم استخدام **Modern OpenGL (3.3 Core Profile)**.
